# stm32-counter-ahitoliev
## Lab assignment with STM32 №2
By: Andrii Ahitoliev

### Конфігурація

Для конфігурування пінів під кнопку та діоди, з допомогою редактора в IDE двоє з пінів
діодів були вибрані як PA1 та PA2, інші - PE7-9 та PE11. Вони введені в режим GPIO_Output.

Зовнішня кнопка - під'єднана до піна PB1 і введена в режим GPIO_Input.

Файл зі схемою є на репозиторії.
